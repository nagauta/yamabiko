# Yamabiko - マイク入力チェッカー

マイクの入力を簡単にチェックできるウェブアプリケーションです。

## 機能

- 🎤 **マイクデバイス選択**: ドロップダウンから使用するマイクを選択
- 📊 **リアルタイム音量表示**: Web Audio APIを使用した音量レベルの可視化
- 🔊 **やまびこ（エコー）機能**: 入力音声をスピーカーから出力
- ⏱️ **遅延時間調整**: 0ms〜2000msまで調整可能なレイテンシ
- 🎨 **モダンなデザイン**: shadcn/uiを使用した美しいUI
- ⚡ **高速動作**: Next.js 14とReact 18による最適化されたパフォーマンス

## 技術スタック

- **フレームワーク**: Next.js 14 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **UIコンポーネント**: shadcn/ui + Radix UI
- **オーディオ処理**: Web Audio API

## セットアップ

### 依存関係のインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてアプリケーションを確認できます。

### ビルド

```bash
npm run build
```

### 本番環境での起動

```bash
npm start
```

## 使い方

1. アプリケーションを開くと、マイクへのアクセス許可を求められます
2. ドロップダウンメニューから使用するマイクデバイスを選択
3. **やまびこ機能**（オプション）:
   - スイッチをオンにすると、マイク入力がスピーカーから出力されます
   - スライダーで遅延時間を調整できます（デフォルト: 300ms）
4. 「テスト開始」ボタンをクリック
5. マイクに向かって話すと、リアルタイムで音量レベルが表示されます
6. やまびこ機能がオンの場合、設定した遅延時間後にスピーカーから音声が再生されます
7. 「停止」ボタンでテストを終了

### 注意事項

- やまびこ機能を使用する際は、ヘッドホンの使用を推奨します（ハウリング防止のため）
- スピーカーで使用する場合は、音量を下げてから試してください

## ブラウザの互換性

このアプリケーションは以下のブラウザで動作します：

- Chrome/Edge (推奨)
- Firefox
- Safari

※ Web Audio APIとMediaDevices APIをサポートするブラウザが必要です
